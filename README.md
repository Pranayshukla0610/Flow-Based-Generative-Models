# Flow-Based-Generative-Models
A comprehensive repository focused on Flow-Based Generative Models and invertible deep learning architectures.  

This repository includes:
- Fundamentals of flow-based models
- Normalizing Flows implementation
- RealNVP and Glow architectures
- Invertible neural networks
- Exact likelihood estimation
- Latent space transformations
- Density estimation techniques
- Advanced generative AI projects

## Tech Stack
- Python
- PyTorch / TensorFlow
- NumPy
- Matplotlib
- Scikit-learn

## Applications
- Image generation
- Density estimation
- Anomaly detection
- Representation learning
- Probabilistic modeling

## Goal
To provide beginner-to-advanced implementations of flow-based generative models for learning, experimentation, and real-world AI applications.
